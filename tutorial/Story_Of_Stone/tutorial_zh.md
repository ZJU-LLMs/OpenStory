# OpenStory 红楼梦互动教程

## 1. 简介

欢迎来到 **OpenStory 红楼梦教程**！本教程将手把手带您“玩转”红楼梦的沙盘世界。从基础的环境搭建到深入的剧情互动，我们为您准备了详尽的步骤解析。无论您是想快速体验经典的红楼故事，还是希望进行二次开发、定制属于自己的专属剧情，本教程都将助您轻松上手。

## 2. 快速开始

在按照快速入门 (QuickStart) 的步骤完成环境配置与 API 设置后，请激活您的虚拟环境，并运行以下命令启动项目：

```bash
python -m examples.story_of_the_stone.run_simulation
```

当您在终端看到以下提示信息时，说明服务已成功启动：
> `API Server started at http://0.0.0.0:8000`

此时，请在浏览器中打开以下链接：

👉 [http://localhost:8000/frontend/index.html](http://localhost:8000/frontend/index.html)

您将看到如下的操作界面：

![操作界面演示](assets/operation.png)

## 3. 核心功能

### 3.1 添加人物

您可以通过以下步骤将新角色引入红楼世界：

1. **点击“添加人物”按钮**，开启角色创建面板。
   ![点击添加人物](assets/add_agent_1.png)
2. **选择角色模板或自定义**：您可以选择官方提供的头像和模板，也可以自行填写全新的人物信息。这里我们以官方提供的“孙悟空”模板为例。
   ![选择角色模板](assets/add_agent_2.png)
3. **确认添加**：信息填写完毕后，点击“确认添加”。
   ![确认添加角色](assets/add_agent_3.png)
4. **添加成功**：此时，您可以看到该角色已经成功降临到红楼梦的世界中啦！
   ![角色添加成功](assets/add_agent_4.png)

### 3.2 指派行动

想要让角色执行特定任务？请在**人物档案**的“指派行动”区域进行设置：

- **计划内容**：可自由发挥，填写您希望角色执行的具体动作。
- **地点**：建议尽量选择地图上已有的地点，以保证互动效果。
- **目标人物**：仅限填写一名目标人物（可以填写地图中尚未出现的全新人物）。

![填写指派行动](assets/order_1.png)

填写完成后，点击 **“下达最高优先级指令”** 按钮。

![下达指令](assets/order_2.png)

> **💡 提示：** 这样，该计划就会被成功添加到角色的行动日程中，在下一 Tick 的推演时，角色将自动执行此指令。

### 3.3 推演与模拟

见证故事发展的时刻到了：

1. **开始推演**：点击页面上的“开始推演”按钮。
   ![开始推演](assets/deduction_1.png)
2. **等待模拟就绪**：系统需要一定时间进行后台计算，请耐心等待，直到“开始模拟”按钮变为绿色。
3. **启动模拟**：点击变绿的“开始模拟”按钮，世界开始运转。
   ![开始模拟](assets/deduction_2.png)
4. **查看成果**：推演结束后，您可以随机点开各个智能体的档案，看看他们在这一轮里都经历了哪些有趣的故事吧！
